## Install

You can install using gradle:

``` setp1:
	allprojects {
		repositories {
			...
			maven { url "https://jitpack.io" }
		}
	}
```

```setp2:
	dependencies {
	        implementation 'com.github.ggbandAdapter:Jsclient:1.0.1'
	}

	
```
